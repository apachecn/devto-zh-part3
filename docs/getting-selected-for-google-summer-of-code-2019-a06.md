# 入选 2019 年谷歌代码之夏

> 原文：<https://dev.to/ravgeetdhillon/getting-selected-for-google-summer-of-code-2019-a06>

今天对我来说是非常特殊的一天。在我的第一次尝试中，我破解了谷歌代码之夏。我很高兴有机会为 GNOME 基金会工作。

我的任务是重建 GTK 网站。对于那些对这个项目的技术感兴趣的人来说，目前的网站是用 PHP 制作的，PHP 是一种很好的网络语言，但是对于创建静态网站来说不是很有用。所以我的工作是从零开始建立一个新的网站，使用内容管理系统的概念。为此，我将使用 Jekyll，网站将使用 Gitlab 的持续集成进行部署。

这将是一个极具挑战性的夏天，但我真的很高兴得到这个机会。这个夏天将增强我对网页设计及其未来的了解。

非常感谢 GNOME 的所有人，是他们选择了我做这份工作。Emmanuele Bassi 将是我今年夏天的导师，我很高兴能从这个人身上学到很多新东西。

对于未来的 GSoCers，这里是我对这个项目的[建议](https://docs.google.com/document/d/1naeFyYH0dLJ30_KcvQes7H4tWI165Xeb5t-Qqfo67NE/edit?usp=sharing):**返工 GTK 网站**。如果你需要任何帮助或者想要我处理你的下一个项目，你可以通过我的[邮箱](//mailto:ravgeetdhillon@gmail.com)或者在 [Instagram](https://instagram.com/ravd_ravgeet/) 上发 DM 给我。
