> 练习触摸打字，在舒适的外壳中与你的朋友竞争，成为一名打字忍者💥。

## 安装

从命令行运行`npm i --global typeracer-cli`

## 特性

*   练习模式(离线模式)
*   用户统计(每分钟字数，所用时间)
*   在线模式(通过生成一个服务器并与你的朋友分享来进行打字比赛)
*   比赛结束后请求重赛(在线模式)
*   可以在在线模式下查看前 10 名高分

## 用法

运行`typerace`或`typerace -h`至其使用位置

**输出**

```
Usage: typerace [options] [command]

  Options:

    -h, --help          output usage information

  Commands:

    practice|p          Start typeracer
    online|o [options]  Start game in online mode 
```
