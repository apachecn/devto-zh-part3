# [T2`tweetfancy.io`](http://tweetfancy.io)

有没有想过用**粗体**、*斜体*或~~删除线~~文本发推文？

Twitter 可能不支持它，但你**可以**。

写在 markdown 和抓住你的花式文字。

###### *注:生成的粗体、斜体和删除线文本为 [UTF 数学字母数字符号](https://en.wikipedia.org/wiki/Mathematical_Alphanumeric_Symbols)。所以如果你用屏幕阅读器来运行它们，你会发现阅读器会根据它们的符号名称来读出它们。但是，所有的屏幕阅读器也很难读懂表情符号(<g-emoji class="g-emoji" alias="tada" fallback-src="https://github.githubassets.cimg/icons/emoji/unicode/1f389.png">🎉</g-emoji> <g-emoji class="g-emoji" alias="fire" fallback-src="https://github.githubassets.cimg/icons/emoji/unicode/1f525.png">🔥</g-emoji> <g-emoji class="g-emoji" alias="rocket" fallback-src="https://github.githubassets.cimg/icons/emoji/unicode/1f680.png">🚀</g-emoji>T9】🦄T11】😎这并不妨碍我们使用表情符号。*
